# Budgeting App (CLI)

## Description
This is a command-line budgeting application built in Python.  
It allows users to:
- Add income
- Add expenses
- View total balance
All transactions are stored in a JSON file for persistent data storage.

## Features
- Add income
- Add expense
- View total balance
- Data persistence using JSON

## Planned Features
- Categorised expenses
- Monthly summaries
- Data visualisation
- Error handling improvements

## Installation & Usage

### 1. Clone the repository
git clone <link>

### 2. Navigate into the folder
cd budgeting-app

### 3. Run the program
python main.py

## Example
Select an option:
1. Add Income
2. Add Expense
3. View Balance

> 1
Enter income amount: 1000
Income added successfully!
